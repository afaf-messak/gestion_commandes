## 📋 Description

**Gestion Commandes** est une application web complète développée avec **Laravel** pour la gestion des commandes clients. Elle permet de gérer les clients, produits, commandes, détails de commandes, statistiques avancées et historique des modifications.

### Fonctionnalités Principales ✨
- ✅ **Gestion CRUD complète** des Commandes, Clients, Produits
- 📊 **Tableau de bord statistiques** (commandes par client/mois, chiffre d'affaires, etc.)
- 👥 **Authentification sécurisée** (login/logout)
- 📈 **Relations Eloquent** (Client → Commandes → Détails → Produits)
- 📝 **Historique des modifications** des commandes (Events/Listeners)
- 🎨 **Interface responsive** Bootstrap 5 + FontAwesome
- 🗑️ **Suppression en cascade** sécurisée
